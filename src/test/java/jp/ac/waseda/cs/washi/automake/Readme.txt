1,ページオブジェクトのみの作成方法
	ページオブジェクトのみの作成の信号(1)
	ページの作成数(int)
	１個目のページの名前(String)
	ページのディレクトリパス
	ページのパッケージ名
	１個目でパースするURL
	２個目のページ名
	２個目のパースするURL
	。。。。
	

2,テストケースのみの作成方法
	テストケースのみ作成の信号（２）
	テストケースの作成数(int)
	１個目のテストケースの名前(String)
	テストケースのディレクトリパス
	テストケースのパッケージ名
	ページオブジェクトのパッケージ名
	テストケース２個目
	テストケース３個目。。。。
	
3,両方作成する場合
	両方作成の信号(3)
	ページの作成数(int)
	テストケースの作成数(int)
	１個目のページの名前
	ページのディレクトリパス
	ページのパッケージ名
	２個目。。。
	
	１個目のテストケースの名前
	テストケースのディレクトリパス
	テストケースのパッケージ名
	２個目。。。。
